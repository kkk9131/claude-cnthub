# 技術的意思決定記録 (ADR)

> このファイルはプロジェクトの重要な技術的決定を記録します。

## 決定一覧

### D001: ランタイムに Bun を採用

- **日付**: 2024-12-27
- **状態**: 採用
- **理由**:
  - Node.js より 3-4x 高速
  - TypeScript ネイティブサポート
  - 組み込みテストランナー
  - claude-mem でも採用実績あり

### D002: Backend フレームワークに Hono を採用

- **日付**: 2024-12-27
- **状態**: 採用
- **理由**:
  - 軽量で高速（Express より高パフォーマンス）
  - TypeScript ファースト
  - Edge Runtime 対応（将来の拡張性）

### D003: データベースに SQLite + sqlite-vec を採用

- **日付**: 2024-12-27
- **状態**: 採用
- **理由**:
  - ローカル専用ツールに適切
  - sqlite-vec でベクトル検索が可能
  - セットアップ不要

### D004: テストフレームワークに Vitest を採用

- **日付**: 2024-12-27
- **状態**: 採用
- **理由**:
  - Bun 互換
  - 高速な実行
  - Vite エコシステムとの親和性
