# /cnthub:approve

提案された失敗セッションを承認し、ネガティブコンテキストを注入する。

## MCP ツール

- `approve_suggest` - 提案を承認

## 使用方法

```
/cnthub:approve        # 全ての提案を承認
/cnthub:approve 1      # 提案1のみ承認
/cnthub:approve 1,3    # 提案1と3を承認
```

## ワークフロー

1. セッション開始時に失敗セッションの提案が表示される
2. ユーザーが `/cnthub:approve` を実行
3. Edge が作成され、ネガティブコンテキストが pending_inject に保存
4. 次のメッセージでネガティブコンテキストが自動注入

## 出力フォーマット

### 承認成功

```markdown
## 承認完了

{N}件の失敗セッションを承認しました:

- {セッション名1}
- {セッション名2}

ネガティブコンテキストが次のメッセージで注入されます。
```

### 提案なし

```markdown
## 提案なし

承認待ちの提案がありません。新しいセッションを開始してください。
```

## エラー

- セッションなし: 「現在のセッションが見つかりません。」
- 提案なし: 「承認待ちの提案がありません。」
- 無効なインデックス: 「有効なインデックスが指定されていません。」
- API エラー: 「提案の承認に失敗しました。」
