# /cnthub:dismiss

提案された失敗セッションを却下し、通知を消去する。

## MCP ツール

- `dismiss_suggest` - 提案を却下

## 使用方法

```
/cnthub:dismiss        # 全ての提案を却下
```

## ワークフロー

1. セッション開始時に失敗セッションの提案が表示される
2. ユーザーが `/cnthub:dismiss` を実行
3. 提案が削除される（Edge は作成されない）

## 出力フォーマット

### 却下成功

```markdown
## 却下完了

失敗セッションの提案を却下しました。
```

### 提案なし

```markdown
## 提案なし

却下する提案がありません。
```

## エラー

- セッションなし: 「現在のセッションが見つかりません。」
- 提案なし: 「却下する提案がありません。」
- API エラー: 「提案の却下に失敗しました。」
