# /review-cc-work - Claude Code の成果をレビュー

Claude Code が完了したタスクをレビューするコマンド。

## 使い方

```
/review-cc-work <タスクID>
```

## 実行内容

1. `cc:完了` マーカーのタスクを検索
2. 変更ファイル一覧を取得
3. テスト結果を確認
4. コードレビューを実施

## レビュー観点

- [ ] 要件を満たしているか
- [ ] テストが通っているか
- [ ] コーディング規約に従っているか
- [ ] セキュリティ問題がないか
- [ ] パフォーマンス問題がないか

## 出力例

```
📋 Review: I-01 サーバー統合

### 変更ファイル
- packages/api/src/index.ts (+45, -12)
- packages/api/src/routes/memories.ts (新規)
- packages/api/package.json (+2, -1)

### テスト結果
✅ 251 tests passed

### レビュー結果
- [x] 要件を満たしている
- [x] テスト通過
- [x] コーディング規約 OK
- [ ] ⚠️ エラーハンドリング追加推奨

### 判定
🟡 条件付き承認（軽微な修正後マージ可）

💡 「承認」で Plans.md を pm:承認 に更新
💡 「修正依頼」で cc:TODO に戻す
```
